# Nmap Automation Script

A simple Python script that automates Nmap scans and saves the output to a timestamped text file.

## Features
- Service and version detection
- Automatic output saving
- Simple command-line usage
- Scan saved to a timestamped file
## Requirements
- Linux
- Python 3
- Nmap

## Usage
```bash
python3 nmap_scan.py 127.0.0.1
